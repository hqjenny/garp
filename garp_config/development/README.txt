
The main subdirectories here:

  gatoconfig        the configurator tool
  ga-emulate        a standalone emulator for a Garp array (without any
                     master processor)
  lib               software libraries for Garp simulation

The libraries under "lib":

  GarpArrayConfig   definitions and function concerning a Garp configuration
  GarpArray         functions for simulating a Garp array
  configCacheTags   functions for simulating the managment of a Garp
                     configuration cache
  cacheTags         functions for simulating the delays of a typical
                     processor cache (not for any Garp cache)
